# 创建虚拟机节点

进入\[节点管理\]页面，点击\[创建VM节点\]，进入创建页面。

![](../../../../../../images/创建节点.jpg)

在页面中选择所需节点配置，填写所需节点带宽大小，系统会为您计算出所选资源的购买价格。

填写节点信息后提交购买，购买成功后返回\[节点管理\]列表。节点资源进入部署中状态。

![](../../../../../../images/返回列表.jpg)

注：
1、申请节点成功后，后端将会收到您的创建申请并为您分配资源。分配时间约为3个工作日，请耐心等待。
2、节点将会在后端为您分配好资源后计费。
3、申请节点时，请保证账户余额大于申请的节点费用，否则将无法创建。
